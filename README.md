## Description of the Project

This project is based on the analysis of the Python programming language as open-source software using Linux shell scripting. The idea behind this project is to understand the working of the Python programming language using the Linux environment and how open-source tools can be used. For this purpose, the Windows Subsystem for Linux (WSL) is used, which can run Linux on the Windows platform without the need to load any other operating system.

## Scripts Included

The project contains 5 shell scripts:

- **Script 1 – System Identity Report**  
   It displays information related to the system, including kernel version, current user, time since system started, and date.

- **Script 2 – Package Inspector**  
   It checks whether Python is installed and displays its version along with a brief description.

- **Script 3 – Disk and Permission Auditor**
   It examines directories like /etc, /home, and /tmp and displays their permission and size.

- **Script 4 – Log File Analyzer**  
   It reads a file and displays how many times the word 'error' is included in it.

- **Script 5 – Manifesto Generator**  
   It takes input from a user and creates a simple open-source statement in a file.
